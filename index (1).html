Cómo usar Content Studio

Primeros pasos

  1. Cargá tus documentos → "Documentos" → subí tu deck, guía de tono o pegá texto.
  2. Configurá las API keys → "Config" → pegá las claves de los modelos.
  3. Elegí el tipo de contenido y completá los campos.
  4. Generá → en segundos tenés hasta 3 variantes listas.

Documentos soportados

PDFDOCXTXTMD

Subí varios a la vez. El sistema extrae el texto, lo combina y se lo pasa a la IA como contexto en cada generación.

Imágenes para posts

  1. Generá el contenido del post.
  2. En cada variante, clic en "Sugerir prompt ✦" → la IA escribe automáticamente la descripción para DALL-E basada en el texto.
  3. Editá el prompt si querés, luego "Generar imagen" (requiere key OpenAI) o "Abrir en DALL-E ↗".

Publicarlo en línea (gratis)

  1. Descargá el archivo index.html.
  2. Creá cuenta en github.com.
  3. Nuevo repositorio → subí el archivo.
  4. Settings → Pages → activar → URL lista en 2 min.
